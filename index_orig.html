R Bootcamp - A Quick Introduction to Reproducible Data Analysis using R

CSU Chico - Math 199 - Fall 2015

Course Information

This course is designed as a primer to get students up and running with the basic knowledge of how to use the statistical programming language R.

Target audience: Anyone who wants to do their own data analysis!
