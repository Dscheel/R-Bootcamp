R Bootcamp

A Quick Introduction to Reproducible Data Analysis using R

Download this project as a .zip file Download this project as a tar.gz file

Purpose

This course is designed as a primer to get students up and running with the basic knowledge of how to use the statistical programming language R. Anyone who wants to do their own data analysis can benefit from this class!
